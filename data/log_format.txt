FORMATO BINARIO LOG GIORNALIERO CRONOTERMOSTATO
================================================

Specifica versione 1.0

HEADER (12 bytes):
------------------
Offset 0-3:   Magic number "TLOG" (0x544C4F47)
Offset 4:     Versione formato (1)
Offset 5-6:   Anno (uint16_t, little endian)
Offset 7:     Mese (1-12)
Offset 8:     Giorno (1-31)
Offset 9-10:  Numero campioni (uint16_t, little endian, max 1440 = 24h x 60min)
Offset 11:    Riservato (0x00)

RECORD (5 bytes per campione):
-------------------------------
Offset 0-1:   Temperatura x10 (int16_t, little endian, es. 205 = 20.5°C)
Offset 2:     Umidità % (uint8_t, 0-100)
Offset 3:     Stato caldaia (uint8_t, 0=OFF, 1=ON)
Offset 4:     Riservato/Flags (uint8_t)

DIMENSIONE FILE:
----------------
Header: 12 bytes
Record: 5 bytes x num_campioni
Totale per 1440 campioni (1 al minuto per 24h): 12 + (5 x 1440) = 7212 bytes

ESEMPIO:
--------
Per 1440 campioni giornalieri (1 sample/minuto):
- File size: 7212 bytes
- Periodo: 24 ore
- Risoluzione: 1 minuto

NAMING CONVENTION:
------------------
log_YYYYMMDD.bin
Esempio: log_20231221.bin
